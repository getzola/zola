Shortcodes in Templates Demo

This page demonstrates the new ability to use shortcodes directly in Tera templates.

1. Simple HTML Shortcode

The note shortcode with different types:

{{ note(text="This is an informational note.", type="info") }} {{ note(text="This is a warning note.", type="warning", title="Warning") }} {{ note(text="This is a danger note.", type="danger", title="Danger") }}

2. Markdown Shortcode (Auto-converted to HTML)

The emphasize shortcode uses markdown syntax:

This text is {{ emphasize(text="really important") }} using a markdown shortcode.

3. Context-Dependent Shortcode

The page_info shortcode accesses page metadata:

{{ page_info() }}

4. Existing YouTube Shortcode

The classic youtube shortcode works in templates too:

{{ youtube(id="dQw4w9WgXcQ", class="responsive-video") }}

5. Conditional Shortcode Usage

You can use shortcodes conditionally:

{% if page.extra.show_note %} {{ note(text="This note is shown conditionally based on page.extra.show_note") }} {% else %}

No note to show (page.extra.show_note is not set)

{% endif %}
